All the unix-commands below assume you are at the root of the submission folder.
It should contain two .java files in it. namely BinaryHeap.java and TestMain.java

Program is tested to work with Java SDK version is 1.8.
To compile:
javac AXE170009/*.java

To run the test cases: (the -ea jvm-option is very important here as the tests use assert for checks)
java -ea AXE170009.TestMain
